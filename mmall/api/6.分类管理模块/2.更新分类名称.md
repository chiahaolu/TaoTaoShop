## 更新分类名称
> 管理员更新分类名称

### 请求说明
> 请求方式：POST<br>
> 请求URL ：`/manage/category/update_category_name.do`

### 请求参数
字段|字段类型|字段说明|是否必传
---|---|---|---
categoryId  |int |分类的id |是
categoryName  |String |新分类名称 |是

### 返回参数
字段  |字段类型   |字段说明
---|---|---
status  |int |0:成功 1:失败
msg  |String |返回消息

### 成功示例
```json
{
    "status": 0,
    "msg": "更新分类名称成功"
}
``` 

### 失败示例
```json
{
    "status": 10,
    "msg": "用户未登录,请重新登录"
}
{
    "status": 1,
    "msg": "无权限操作,需要管理员权限"
}
{
    "status": 1,
    "msg": "更新分类名称失败"
}
{
    "status": 1,
    "msg": "该分类不存在,添加后才可修改"
}
```
