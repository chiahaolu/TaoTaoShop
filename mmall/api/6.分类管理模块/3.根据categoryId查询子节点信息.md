## 根据categoryId查询子节点信息
> xxxxx

### 请求说明
> 请求方式：GET<br>
> 请求URL ：`/manage/category/get_category.do`

### 请求参数
字段|字段类型|字段说明|是否必传
---|---|---|---
categoryId  |int |分类节点id |是

### 返回参数
字段  |字段类型   |字段说明
---|---|---
status  |int |0:成功 1:失败
msg  |String |返回消息
data    |Object |返回的数据
id  |int    |分类节点id
parentId    |int    |分类节点的parentId
name    |String |分类节点名称
status  |bool   |分类节点状态(true:可用;false:不可用)
sortOrder   |   |分类节点排序
createTime  |timestamp  |创建时间
updateTime  |timestamp  |更新时间
