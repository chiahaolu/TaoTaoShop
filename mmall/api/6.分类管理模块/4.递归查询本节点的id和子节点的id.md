## 递归查询本节点的id和子节点的id
> xxxxx

### 请求说明
> 请求方式：GET<br>
> 请求URL ：`/manage/category/get_deep_category.do`

### 请求参数
字段|字段类型|字段说明|是否必传
---|---|---|---
categoryId  |int |分类节点id |是

### 返回参数
字段  |字段类型   |字段说明
---|---|---
status  |int |0:成功 1:失败
msg  |String |返回消息
data    |Object |返回的数据
