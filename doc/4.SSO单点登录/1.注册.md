---|---|---|---

### 请求说明
> 请求方式：POST<br>
请求URL ：`http://localhost:8084/user/register`

### 请求参数
字段|字段类型|字段说明
---|---|---
username  |String |用户名
password  |String |密码
phone  |String |手机号
email  |String |邮箱
question  |String |找回密码问题
answer  |String |找回密码答案

### 返回参数
字段  |字段类型   |字段说明
---|---|---
status  |int |状态码
msg  |String |信息
data  |String |token信息


### 返回结果
```json
{
    "status": 200,
    "msg": "OK",
    "data": "5e26b825-3966-495f-9daf-a5f677542b42"
}
{
    "status": 500,
    "msg": "用户名或密码为空",
    "data": null
}
```
