### 请求说明
> 请求方式：POST<br>
请求URL ：`http://localhost:8084/user/check/{param}/{type}`

### 请求参数
字段|字段类型|字段说明
---|---|---
param  |String |需要校验的数据内容
type  |int |数据类型(1:用户名 2:手机号 3:email)
callback  |String |回调地址(可选)

### 返回参数
字段  |字段类型   |字段说明
---|---|---
status  |int |状态码
msg  |String |消息
data  |boolean |是否可用


### 返回结果
```json
{
    "status": 200,
    "msg": "OK",
    "data": true
}
{
    "status": 200,
    "msg": "OK",
    "data": false
}
```
